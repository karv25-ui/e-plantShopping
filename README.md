Paradise Nursery - Houseplant E-commerce Application
Paradise Nursery is a premium React-based web application designed for houseplant enthusiasts. This project was developed as part of the IBM React Assignment, focusing on state management with Redux, dynamic routing, and premium UI/UX design.
